## [1.0.0] - 2025-10-15

### Added

* Initial release
* Climate entity support for Duux Edge heaters
* Temperature control (5-36°C)
* Three preset modes (Low, Boost, High)
* Night mode support
* Auto-discovery of devices
* HACS support
* Configuration flow via UI

### Known Issues

* None
